# Clinic-Appointment-and-Diagnostics-Platform

This project contains an ER diagram for a simple clinic management system.

ER Diagram:
<img width="1833" height="1366" alt="Clinic Appointment and Diagnostics Platform dark" src="https://github.com/user-attachments/assets/a853cf77-e2fa-4578-b07f-e041785e10f5" />
Eraser link: https://app.eraser.io/workspace/1WoZ2zNgU0rsKD42UVvf?origin=share

It supports:
- Managing doctors and patients  
- Booking appointments  
- Recording consultations (actual visits)  
- Prescribing tests and generating reports  
- Managing prescriptions and payments  

Key points:
- Appointment is just a booking, consultation is the actual visit  
- Tests are prescribed during consultation and stored separately  
- Reports are linked to prescribed tests  
- Patient health data is stored per consultation, not in the patient table  

The ER diagram image is included in this repository.
